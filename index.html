기획서

기획 목적
  • 자격증 / 공인어학시험 등 일정이 정해진 시험을 목표로 공부하는걸 돕는 웹/어플리케이션 서비스
기능 정리
  • 실 공부 시간 등록
    • 시간 입력(공부 시작, 종료, 휴식 등)
    • 공부 시간/ 휴식 시간 지정하기
    • 휴식 시간 알림
  • 플래너
    • 공부계획(월/주/일)
    • 공부 목표에 대한 정보 설정
    • 계획 실천 여부 체크하는 것
  • 단어장(개념정리)
    • 어학 단어/전문 용어 정리
    • 퀴즈 기능
  • 분석
    • 공부시간 평균 내기(주/전체)
    • 계획과 실공시간의 일치 여부 분석
    • 퀴즈 정답률 분석
사용 기술
